## Usage
1. Clone this repository
```bash
git clone https://github.com/kikuKeii/rumah-sehat.git
```
2. Install packages (use `yarn` or `npm`)
```bash
npm install 
```
or
```bash
yarn install
```
3. Start server
```bash
npm next dev
```
or
```bash
yarn next dev 
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
